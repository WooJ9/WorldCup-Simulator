Group1_WorldcupSimulator.py를 실행하여 전체 실행결과를 볼 수 있습니다.

fifaova.csv는 fifa-game stat데이터 파일입니다.

matches.csv는 2007년 이후 국제경기 결과 데이터 파일입니다.

Group1_updated_data.csv는 모델에 실제로 사용되는 main 데이터 파일입니다.

layers.py, DNN.py, functions.py, trainers.py, optimizer.py은 Group1_WorldcupSimulator에서 import 되는 파일들입니다.

preprocessing.py는 fifaova와 maches를 전처리 시켜주는 파일입니다.
